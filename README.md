# Finance-Logger

## 📘 Description

**Finance-Logger** helps users easily record and organize their financial transactions in a clean, browser-based interface.  
You can add new invoices and payments, view a complete list of records, and manage your logs with a clear and minimal UI.

The app is built with **TypeScript** to provide strong typing, better structure, and improved maintainability.  
It showcases an object-oriented approach with reusable components and direct DOM manipulation for dynamic rendering.

## ✨ Features

- Add and manage invoices and payments
- Display all transactions dynamically in the browser
- Reusable and modular `ListTemplate` class for rendering items
- Separation between `src/` (TypeScript source) and `public/` (compiled JavaScript)
- Clean and responsive UI

## 🧠 Tech Stack

- **TypeScript** — for structure and type safety
- **JavaScript (compiled output)** — runs in any modern browser
- **HTML5 & CSS3** — for layout and styling
